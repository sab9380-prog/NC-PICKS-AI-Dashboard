핵심 설계 변경

  • SPI (Schedule Performance Index) — 실제진척/계획진척. 1.0 = 정상, <0.7 = 위험(빨강), 0.7~0.9 = 주의(노랑). 주관 배제, 숫자로 판단
  • "즉시 주의 필요" 영역 — 위험+주의 시스템만 상단에 별도 하이라이트. 경영진이 스크롤 없이 문제 시스템 즉시 파악
  • 계획 vs 실제 비교 바 — 회색 마커(계획 위치) vs 색상 바(실제 진척). 차이가 클수록 빨간색
  • 지연 일수 — "38일 지연"처럼 구체적 숫자. "지연" 배지보다 훨씬 명확
  • 트렌드 화살표 — ↗(개선) →(정체) ↘(악화). 지난주 대비 방향성
  • 위험도순 정렬 — SPI 낮은 순서. 정상 시스템은 하단으로 접힘 가능

SPI 계산 방식

경과율 = (오늘 - 시작월) / (목표월 - 시작월)
진척률 = 현재 점수 / 100
SPI = 진척률 / 경과율

예: 시작 26.01, 목표 26.12, 현재 26.04 → 경과율 27%
현재 점수 40pt → 진척률 40% → SPI = 40/27 = 1.48 (초과달성)
현재 점수 10pt → 진척률 10% → SPI = 10/27 = 0.37 (심각 지연)
